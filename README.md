# EcosystemsAllEngines

Meta-gem that bundles all ecosystem engine gems for convenient inclusion.

## Installation

Add to your Gemfile:

```ruby
gem 'ecosystems-all-engines'
```

## Included Gems

- **rayswarm-core** — AI chat interface, intention detection, tab registry, domain engine registry (Layer 1.0)
- **rayswarm-design-system** — Design system engine
- **rayswarm-medallion** — Medallion architecture engine
- **rayswarm-platform** — Platform engine
- **rayswarm-scoping** — Scoping engine
- **rayswarm-service-provider** — Service provider engine
- **service-protege** — Bidirectional OWL/RDF ontology interchange with LLM analysis (Layer 2.0)
