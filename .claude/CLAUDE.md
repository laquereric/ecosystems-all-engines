# service-all-engines

Meta-gem that bundles all service-ecosystem engine gems.

## Engines

### engine/ (Rails Engines)
- **rayswarm-core** (`engine/service-core`) - Core engine
- **service-protege** (`library/service-protege`) - Protege engine

### container/ (Mountable Rails Engines)
- **service-3d** (`container/service-3d`) - 3D rendering engine
- **service-id** (`container/service-id`) - Identity/verification engine
- **service-mermaid** (`container/service-mermaid`) - Mermaid diagram engine
- **service-puml** (`container/service-puml`) - PlantUML diagram engine
- **service-switch** (`container/service-switch`) - Switch engine
- **service-rubygems** (`container/service-rubygems`) - RubyGems engine
- **service-github** (`container/service-github`) - GitHub integration engine

## Current Dependencies

This gem currently depends on:
- `rayswarm-core`
- `service-protege`
